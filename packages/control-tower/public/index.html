Session waterfall

Every workflow and subagent here was launched by the main session (the conversation). Bars are placed on a shared time axis by real start/end timestamps; x-position ≈ when each was initiated, length ≈ duration. Cost is a cache-aware estimate.

Control (shim): pick a bundled workflow, then edit its agents' models and prompts below and press Run Workflow — the edited version runs through the standalone workflow-lens runtime, outside the Claude Code harness, under your controls: a Budget Cap, Cost Router (incl. non-Anthropic via OpenRouter), a Cache/HITL gate, or deterministic Replay. Edits splice the original source in place, so loops are preserved; agents whose prompts are built at runtime are shown read-only. This turns a Claude-authored workflow file into an editable, portable agent executor.
